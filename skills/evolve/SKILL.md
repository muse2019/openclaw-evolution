# OpenClaw 自动进化

## 风险等级

### 🟢 自动执行（无需确认）

**条件**: 变更不改变语义，仅改进表达

以下变更会自动执行：
- 技能文件中的拼写错误修正
- 指令措辞的清晰化改进
- 错误消息的改进
- 格式化问题修复

### 🟡 询问后执行（需用户确认）

**流程**: 显示变更预览 → 用户选择: 批准 / 拒绝 / 修改

以下变更会显示预览并询问用户：
- 新增技能或工具
- 修改现有指令内容
- 更新配置参数
- 添加新的记忆或习惯

### 🔴 禁止自动执行（仅生成报告）

**行为**: 生成 Markdown 报告文件，用户自行决定是否采纳

以下变更只生成建议报告：
- 删除技能或工具
- 改变核心行为模式
- 重构框架结构
- 修改安全相关设置
- API 密钥、密码等敏感数据

## 触发机制

### 错误驱动

当满足以下条件时自动触发：
- 最近 24 小时内错误数 ≥ 3
- 同类错误重复出现
- 距上次触发已过冷却期 (30 分钟)

### 定时评估

默认每 24 小时运行一次：
- 分析所有错误模式
- 评估框架结构
- 检查效果指标
- 生成改进提案

### 手动触发

用户可随时触发：
- `/evolve` - 标准进化周期
- `/evolve --report` - 仅生成报告（所有🟢🟡变为🔴）
- `/evolve --full` - 完整分析
- `/evolve --target X` - 针对特定目标

## 安全机制

### 路径限制

**允许修改**:
- `~/.openclaw/workspace/skills/**`
- `~/.openclaw/config/preferences.json`
- `~/.openclaw/memory/**`

**禁止修改**:
- `~/.openclaw/auth/**`
- `~/.openclaw/secrets/**`
- `.env` 文件
- 密钥文件

### 频率限制

| 等级 | 每小时 | 每天 |
|------|--------|------|
| 🟢 自动 | 10     | 50   |
| 🟡 询问 | 5      | 20   |
| 🔴 禁止 | 20     | 100  |

### 回滚机制

每次进化都保存变更前快照，支持：
- 回滚单个进化
- 回滚最近 N 个进化
- 回滚指定时间点之后的所有进化

## 工具定义

### evolve

触发进化周期

参数:
- `reportOnly` (boolean): 仅生成报告（不执行🟢🟡）
- `target` (string): 针对特定目标
- `fullAnalysis` (boolean): 运行完整分析

### evolution_rollback

回滚进化

参数:
- `evolutionId` (string): 进化 ID
- `reason` (string): 回滚原因

### evolution_status

获取引擎状态

### evolution_history

获取进化历史

参数:
- `count` (number): 返回记录数量
